D'après ce qui précède, on déduit qu'une opération (expression) soumise par l'utilisateur ne peut contenir que 5 types d'éléments :
- Les Nombres(qui sont forcément positifs),
- Les Opérateurs,
- Les Constantes,
- Les Séparateurs et
- Les Fonctions externes.
